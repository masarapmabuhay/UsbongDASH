As long as your Android device is connected to the internet, you can download Usbong trees (i.e. .utree and .xml files) by pressing its File Name (e.g. promeng.utree). The file will be downloaded to your Android device's Download folder (e.g. </storage/emulated/0/>Download). Using your file browser (e.g. File Manager app), copy-paste it to /usbong/usbong_trees. If you're using the File Manager app, to copy a file, press the "Multi" button on top, press your target file, and then press "Copy". Go to your target directory, and then press the "Paste" button which is on the left of the "Home" button. Happy Tree Planting!
